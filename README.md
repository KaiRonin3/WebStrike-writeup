## SOC Investigation: WebStrike

A structured incident report produced from analysing a packet capture of a real web
server compromise. The investigation traces the full attack chain - from initial
reconnaissance through file upload bypass, reverse shell deployment, and exfiltration
attempt - using Wireshark.

Covers CWE-434 (unrestricted file upload), double extension bypass technique, reverse
shell mechanics, and MITRE ATT&CK mapping across five tactics. Written in the format
of a report a Tier-1 analyst would submit to their team lead.
